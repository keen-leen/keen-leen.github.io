# 戳气球(力扣 312)

---

<aside>
💡

有 `n` 个气球，编号为`0` 到 `n - 1`，每个气球上都标有一个数字，这些数字存在数组 `nums` 中。

现在要求你戳破所有的气球。戳破第 `i` 个气球，你可以获得 `nums[i - 1] * nums[i] * nums[i + 1]` 枚硬币。 这里的 `i - 1` 和 `i + 1` 代表和 `i` 相邻的两个气球的序号。如果 `i - 1`或 `i + 1` 超出了数组的边界，那么就当它是一个数字为 `1` 的气球。

求所能获得硬币的最大数量。

</aside>

### 思路

1. 常规思路是枚举每个气球，计算逐个戳破可以获得的最大硬币。但是在戳破的过程中其他气球的排列状态也在发生变化，维护起来比较困难。
2. 因此可以采取逆向思维，枚举最后一个戳破的气球，此时所有气球的状态都已确定，即该气球的左右气球都可以确定。

### 实现

1. dp[i][j] 表示戳破开区间 (i, j) 内的所有气球可以获得的最大硬币数；
2. j - i ≤ 1 时，dp[i][j] == 0，因为开区间内没有任何气球；
3. 从小到大枚举所有区间 (i, j)，在区间内枚举最后一个戳破的气球 k；
4. 最后一个戳破 (i , j) 中的气球 k 时，k的左右相邻气球都可以确定（i 和 j），因此戳破气球 k 带来的收益也可以确定 nums[i] * nums[k] * nums[j]；
5. dp[i, k] 和 dp[k, j] 也在从小到大的区间计算中算出（因此一定要保证区间从小到大计算），因此可以动态的确定当前区间能获得的最大硬币数；
6. 最后 dp[0][n + 1] 即为所求值（为方便计算将nums左右两侧都加一个元素并置为 1）

### 代码

```java
class Solution {
    public int maxCoins(int[] nums) {
        if (nums == null || nums.length == 0) {
            return 0;
        }
        int n = nums.length;
        int[] val = new int[n + 2];
        val[0] = val[n + 1] = 1;
        for (int i = 1; i <= n; i++) {
            val[i] = nums[i - 1];
        }
        // dp[i][j]表示戳破开区间(i, j)内气球能获得的最大硬币数
        int[][] dp = new int[n + 2][n + 2];
        for (int i = n - 1; i >= 0; i--) {
            for (int j = i + 2; j <= n + 1; j++) {
                // 枚举(i, j)中的每一个位置（最后戳破的气球位置）
                for (int k = i + 1; k < j; k++) {
                    int sum = dp[i][k] + val[i] * val[k] * val[j] + dp[k][j];
                    dp[i][j] = Math.max(dp[i][j], sum);
                }
            }
        }
        return dp[0][n + 1];
    }
}
```
